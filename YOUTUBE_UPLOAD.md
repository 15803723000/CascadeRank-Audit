## Title

CascadeRank Audit: Evidence-First GNN Ranking Claims with Codex

## Description

CascadeRank Audit is a local, evidence-first Codex plugin for testing graph
critical-node ranking claims. It detects label-feature leakage, compares a GNN
against six non-learned baselines, runs repeated node-removal experiments, and
produces a fixed claim ledger with independently verified artifact hashes.

The project was built with Codex and GPT-5.6. Codex helped implement the audit
protocol, verifier, tests, plugin packaging, reproducible demo, and release
workflow. The demo preserves negative results instead of converting them into
an unsupported model-success narrative.
